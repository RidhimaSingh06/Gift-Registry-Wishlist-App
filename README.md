# Gift-Registry-Wishlist-App
A personal gift registry and wishlist platform where users can create wishlists and allow friends to reserve gifts to avoid duplicates.

Description

This project is a web application that allows users to create and share wishlists for events like birthdays, weddings, or festivals. Friends and family can view the wishlist and reserve gifts to avoid duplicate purchases.

Technologies Used
React.js
Node.js
Express.js
MongoDB
JWT Authentication
Features
User Registration & Login
Create Wishlist
Add Gift Items
Share Wishlist Link
Reserve Gifts
Prevent Duplicate Reservations
Installation
npm install
npm run dev
